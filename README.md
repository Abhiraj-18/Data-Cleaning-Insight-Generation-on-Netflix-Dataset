# Data-Cleaning-Insight-Generation-on-Netflix-Dataset


## 📌 Project Overview
This project focuses on cleaning, transforming, and analyzing a real-world Netflix dataset to prepare it for analysis and generate meaningful business insights. The objective was to improve data quality and perform exploratory data analysis (EDA) using Excel and Power BI.

---

## 🎯 Objectives
- Clean and standardize raw Netflix data
- Handle missing, null, and duplicate values
- Prepare the dataset for analysis and visualization
- Identify trends in content type, release years, and genres

---

## 🛠 Tools & Technologies
- Excel
- Power BI
- Power Query

---

## 🔄 Data Cleaning & Preparation Steps
- Imported CSV dataset into Excel and Power BI
- Used first row as headers and standardized column names
- Identified and handled missing values:
  - Replaced missing director names with “Director Missing”
  - Replaced missing cast values with “Cast Missing”
- Removed empty rows and duplicate records
- Standardized date formats and categorical values
- Validated data consistency using Power Query profiling tools

---

## 📊 Exploratory Data Analysis (EDA)
- Analyzed content distribution across Movies and TV Shows
- Identified trends by release year
- Explored genre-wise and country-wise content distribution
- Evaluated rating patterns across different content types

---

## 📈 Key Insights
- Movies dominate the Netflix catalog compared to TV shows
- Significant growth in content releases after 2015
- Drama and International content are among the most common genres
- Data quality improvements significantly enhanced analysis accuracy

---

## 🚀 Outcome
- Delivered a clean, analysis-ready dataset
- Improved data accuracy and consistency
- Built a strong foundation for dashboarding and business insights
